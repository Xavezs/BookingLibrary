1. Login Admin

1
Buka aplikasi BookWorm melalui domain/local URL yang disediakan.
2
Pada halaman login, isi Username dengan admin dan Password dengan 123123.
3
Klik tombol Login. Jika berhasil, aplikasi akan masuk ke Librarian dashboard.
Catatan validasi: Jika username/password salah, sistem menampilkan pesan error dan tidak membuat sesi login.

4. Kelola Member

1
Buka menu Members.
2
Klik Add Member untuk membuat akun member baru.
3
Isi Full Name, Username, Password, Email, Member ID, dan Status.
4
Klik Save Member. Akun login member otomatis dibuat.
FieldAturan
UsernameHarus unik. Digunakan untuk login member.
PasswordMinimal 6 karakter.
EmailFormat email valid dan unik.
Member ID10 karakter, diawali angka 2.
StatusStudent atau Faculty. Student 14 hari masa pinjam, Faculty 21 hari.
Delete member: Saat member dihapus, user login terkait juga ikut dihapus sehingga akun lama tidak bisa login lagi.

5. Kelola Library Loan

Menu Library Loan adalah pusat transaksi reservation, checkout, return, renew, dan fine testing.

StatusArtiAction Admin
ReservedMember sudah booking buku, tetapi buku belum resmi dipinjam.Check-Out atau Cancel.
BorrowedBuku sudah dipinjam dan sedang berada di member.Check-In, Renew, atau Test Fine.
ReturnedBuku sudah dikembalikan.Tidak ada action lanjutan kecuali fine payment oleh member.

Check-Out

1
Buka Library Loan.
2
Cari transaksi dengan status Reserved.
3
Klik Check-Out. Status berubah menjadi Borrowed.

Check-In

Klik Check-In untuk mengembalikan buku. Book status akan kembali Available.

Renew

Klik Renew untuk memperpanjang due date sesuai status member.

Cancel

Klik Cancel hanya untuk transaksi Reserved. Book status kembali Available.

6. Fine Rate dan Denda

Daily fine rate berada di panel Quick Review pada Dashboard Admin.

1
Masukkan nominal denda per hari, misalnya 2500.
2
Klik tombol centang Save.
3
Untuk demo denda cepat, buka Library Loan pada loan Borrowed lalu klik Test Fine.
Batas sistem: Fine rate maksimal Rp 100.000 per hari supaya nominal denda tidak merusak demo.
