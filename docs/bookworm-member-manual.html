3. Cari dan Reserve Buku

1
Klik menu Books/Cataloging.
2
Gunakan search box untuk mencari berdasarkan title, author, atau ISBN.
3
Pilih buku dengan status Available.
4
Klik Reserve, lalu klik Confirm pada modal konfirmasi.
5
Sistem menampilkan receipt reservasi berisi title, author, borrower email, borrower number, borrow date, dan due date.
Business rule: Member tidak bisa membuat reservasi baru jika masih memiliki unpaid fine.

4. Melihat Status Pinjaman

Buka menu Library Loan. Tabel ini menampilkan riwayat dan status transaksi member.

StatusArti untuk MemberAksi yang Tersedia
ReservedBuku sudah dibooking, menunggu admin melakukan Check-Out.Tunggu librarian.
BorrowedBuku resmi sedang dipinjam.Return Book. Jika overdue, bisa muncul Pay Fine.
ReturnedBuku sudah dikembalikan.Completed atau Pay Fine jika masih ada denda.

5. Return Book

1
Buka Library Loan.
2
Cari transaksi dengan status Borrowed.
3
Klik Return Book.
4
Jika buku terlambat, sistem menghitung denda berdasarkan due date dan daily fine rate.

6. Top Up dan Bayar Denda

Top Up Wallet

1
Pada Dashboard atau Library Loan, klik tombol Top Up.
2
Saldo wallet bertambah untuk kebutuhan demo pembayaran denda.

Bayar Fine

1
Buka Library Loan.
2
Jika ada fine dengan badge Unpaid, klik Pay Fine.
3
Jika saldo cukup, sistem memotong wallet dan menampilkan payment receipt.
KondisiHasil
Wallet cukupFine berubah Paid, wallet berkurang, unpaid fine menjadi 0.
Wallet tidak cukupSistem menampilkan pesan untuk top up terlebih dahulu.
Tidak ada fineKolom fine menampilkan tanda minus dan tombol Pay Fine tidak muncul.
